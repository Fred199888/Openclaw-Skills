# OpenClaw 定时任务教学文档（Cron）

## 一、你需要知道的目录
把本包里的 `.openclaw` 拖到新电脑的用户主目录下（与现有 `.openclaw` 合并）：
- `~/.openclaw/config.json` / `openclaw.json`：主配置
- `~/.openclaw/skills/*`：自定义技能
- `~/.openclaw/cron/*`：定时任务持久化数据（若存在）

## 二、创建提醒（推荐方式）
在聊天里直接说：
- “30分钟后提醒我开会”
- “每天早上9点提醒我看日报”

OpenClaw 会通过 cron 工具创建任务。

## 三、常见调度类型
1. **一次性任务（at）**
- 指定绝对时间触发一次。

2. **间隔任务（every）**
- 每隔 N 毫秒运行一次。

3. **Cron 表达式（cron）**
- 例如：`0 9 * * 1-5`（工作日 9:00）

## 四、任务执行模式
- `sessionTarget=main` + `payload.kind=systemEvent`
  - 把提醒注入主会话（最常见）
- `sessionTarget=isolated` + `payload.kind=agentTurn`
  - 独立子会话执行复杂定时任务

## 五、迁移到新电脑
1. 安装并运行 OpenClaw
2. 退出 OpenClaw（避免写冲突）
3. 把本包内 `.openclaw` 目录拖拽覆盖到新电脑的 `~/.openclaw`
4. 启动 OpenClaw
5. 在会话中验证：
   - 自定义 skill 能被识别
   - 定时任务可正常触发

## 六、排查建议
- 若 skill 不生效：检查目录层级必须是 `~/.openclaw/skills/<skill-name>/SKILL.md`
- 若定时任务不触发：检查 OpenClaw 是否在线、时区是否正确、任务是否启用
- 若配置冲突：先备份旧 `.openclaw`，再按需合并 `config.json/openclaw.json`
