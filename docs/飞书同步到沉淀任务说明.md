## 迁移后检查
1. 确认目标机技能路径：`~/.openclaw/skills/<skill>/SKILL.md`
2. 确认 cron 文件存在：`~/.openclaw/cron/jobs.json`
3. 启动 OpenClaw 后检查：
   - 定时任务是否在列表中
   - 到点是否投递到 Feishu（delivery channel）

## 注意
- 若目标机账号/凭据不同，Feishu 投递目标可能需要重配。
- 若脚本绝对路径变化，请在 cron 任务 message 中改成目标机真实路径。
